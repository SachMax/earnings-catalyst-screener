# Using sys Module

## `sys.platform`
A string contains a platform identifier
| System | platform value |
| Linux | 'linux' |
| Windows | 'win32' |
| macOS | 'darwin' |

## `sys.copyright`
A string containing the copyright of the Python interpreter

## `sys.version`
A string containing the version number of the Python interpreter, plus additional information of the compiler

## sys.stdout.write("String")
Standard out is used to output a statement

## sys.stderr.write("String")
Output an error text with red color

## `sys.getsizeof(variable)`
gets the size of the variable in bytes

## `sys.argv`
to return list of command line argumnents passes to a python script
